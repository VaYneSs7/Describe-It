# Describe-It
Describe-It - is a local web server game, where you teach English by describing random words. Written in C++ (used Crow and SQLite libraries) and CSS (for web page)

-Powered by a SQLite database with 466K+ words.
-Modern nice design
-Every description you write is saved to a local database, not your browser cache.

Just run .exe file and browser will open automatically at "http://localhost:18080".
